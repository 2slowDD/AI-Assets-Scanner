Design notes

1
Constrained width: The card is max 920px wide and left-aligned — on 1440px+ screens it won't stretch to fill the frame.
2
Sonar icon: Static SVG radar in admin menu sidebar and in the card header. Animated (rotating sweep + pulsing rings) shown only during discovery loading.
3
Plugin notices: Auto-bypass notices are compact — single line with icon instead of full WP notice blocks. Less visual noise.
4
URL groups: Pages / Posts / Other (custom post types). Each group has a header checkbox (select/deselect all in that group) + individual checkboxes per row. Zebra stripes alternate light/white.
5
Filter pills: All / Pages / Posts / Other — toggles which group is visible. "Select all" / "Deselect all" operate on the currently filtered set (or all if "All" filter active).
6
Exclusion textarea: Kept as a secondary method. A hint line explains that deselecting above is the simpler alternative.
7
Credit badge: Updates live as you deselect URLs — shows count of deselected in grey parenthetical.
8
Step progress: Four dots in the header, colour-coded: blue = current, green = done, grey = upcoming.
