# Security Policy

## Supported Versions

Only the latest released version of AI Assets Scanner is actively supported for security fixes. Older versions may be asked to upgrade before a fix is provided.

## Reporting a Vulnerability

Please do not open a public GitHub issue for security reports.

Report suspected vulnerabilities through:

- https://wpservice.pro/contact/

Include the affected version, a clear description, reproduction steps, expected impact, and any proof-of-concept details needed to validate the issue.

## Scope

This policy covers the AI Assets Scanner WordPress plugin code in this repository. Private SaaS, licensing, deployment, and update-service infrastructure are not in scope unless you have written authorization.

## Handling

We aim to acknowledge valid reports within 3 business days, validate the issue, prepare a fix when needed, and coordinate disclosure after affected users have had a reasonable chance to update.

Please keep vulnerability details confidential until a fix or mitigation is available.
